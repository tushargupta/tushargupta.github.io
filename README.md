## About

This is a static blog built with HTML, CSS, and JavaScript, hosted on GitHub Pages. It focuses on:

- **Technology & AI** - Platform economics, AI trends, and their impact on work
- **Technical Writing** - Documentation architecture, DITA, and content systems at scale
- **Career Insights** - Workplace dynamics and navigating the tech industry
- **Personal Reflections** - Thoughts on productivity and the human side of technology

---

## Technical Stack

- **Frontend:** Pure HTML5, CSS3, JavaScript (no framework)
- **Hosting:** GitHub Pages
- **Analytics:** Google Analytics 4
- **Design:** Responsive, mobile-first design
- **Components:** Modular HTML includes for header, sidebar, and common elements
